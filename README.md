# ⚡ Admin Quick Actions by YkigaiCode

Spotlight-style command palette for the WordPress admin.  
Search content, navigate faster, and trigger actions instantly — without clicking through endless menus.

---

## ✨ Features

- ⌘ / Ctrl + K command palette
- Deep admin menu search (includes submenus)
- Instant search for posts, pages, and WooCommerce data
- Recent items
- Open results in a new tab (Cmd/Ctrl + Enter)
- Keyboard-first workflow
- Lightweight & fast
- No external services — 100% local

---

## ⌨️ Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
Open palette (Mac) | `Cmd + K`  
Open palette (Windows/Linux) | `Ctrl + Shift + K`  
Fallback | `Alt + K` or `Ctrl + Space`  
Open in new tab | `Cmd/Ctrl + Enter`  

---

## 🧠 Why?

The WordPress admin is powerful — but slow to navigate.

This plugin brings a **Spotlight / Raycast / VSCode-style workflow** to wp-admin so you can:

- jump between screens
- create content
- manage WooCommerce
- access settings

⚡ in seconds.

---

## 📦 Installation

1. Upload the plugin to `/wp-content/plugins/`
2. Activate it from the **Plugins** screen
3. Press your shortcut and start typing
