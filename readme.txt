== Description ==

**Admin Quick Actions by YkigaiCode** adds a lightweight command palette to your WordPress admin.

Open it with:
- **Cmd + K** (Mac)
- **Ctrl + Shift + K** (Windows/Linux)
- Fallback: **Alt + K** or **Ctrl + Space**

Then type things like:
- `new page`
- `menus`
- `settings`
- `Woo orders` (when WooCommerce is active)
- Any page/post/product title to jump straight into editing

It also includes a help page under **Dashboard → Quick Actions by YkigaiCode** — by YkigaiCode.

== Features ==

- Spotlight-style command palette inside wp-admin — by YkigaiCode
- Deep **Admin Menu** search (including submenu items)
- **Recent items** (auto-saved)
- **Cmd/Ctrl + Enter** opens results in a new tab
- Content search (Pages/Posts + Products/Orders with WooCommerce and permissions)
- Dashboard widget + Dashboard help page

== Installation ==

1. Upload the ZIP in **Plugins → Add New → Upload Plugin**
2. Activate **Admin Quick Actions by YkigaiCode**
3. Press **Cmd+K** (Mac) or **Ctrl+Shift+K** (Windows/Linux)
4. Visit **Dashboard → Quick Actions by YkigaiCode** for tips — by YkigaiCode

== Frequently Asked Questions ==

= Why doesn’t Ctrl + K work? =
Many browsers reserve **Ctrl + K** for the browser search/address bar. Use **Ctrl + Shift + K**, **Alt + K**, or **Ctrl + Space** instead — by YkigaiCode.
